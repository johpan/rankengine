how to use RankEngine

Note: This app requires Google Sheets.
  1. Make a copy of this template sheet and make changes.
  2. After you've filled it in, publish it.
    File => Publish to the web
  3. Click on the config button and paste in your rankings sheet hash.
  4. If your ranking isn't the first sheet in the spreadsheet, input the sheet number.
  5. Click Submit.
